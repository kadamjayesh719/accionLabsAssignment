Developing Environment(required)
	Angular JS v1.5.x
	Node JS  v7.10.1
	Express JS v4.16.2
	Mysql v5.6.37
	MongoDB v3.4.7

Please Follow The Steps Below

	1.Run npm install to download all the dependencies
	2.Run firstRun.sql file in your mysql console
	3.configure config.js for mysql_username & mysql_passsword
	4.By default 0 is mentioned in config.js for dbChoice which will consider MySql as database please change it to 1 for mongoDb
	5.run node app.js to start server
	6.check http://localhost:4000 for application
